  2. Подключились

  3. С помощью команды ping на учебном сервере узнать IP-адрес веб-сервера kubsu.ru.

  4. С помощью команды nslookup узнать A-записи и MX-записи домена kubsu.ru и kubsu-dev.ru

  5. С помощью команды whois узнать дату регистрации домена kubsu.ru и kubsu- dev.ru

  6. С помощью SSH склонировать репозитарий со скриншотами и страницей в каталог www. Веб-страница должна открываться по адресу http://ваш_логин.kubsu-dev.ru/каталог_задания_1/

  7. С помощью программы FileZilla соединиться с учебным сервером с вашим логином и паролем по протоколу SFTP и скопировать на локальный компьютер файлы задания из каталога www.
